### How to pass a component as props

1. Import the desired component at the top of your script-tag.

```javascript
import PubButton from '@/publiq-ui/pub-button.vue';
```

2. Create a computed value that returns the imported component.

```javascript
computed: {
  pubButton() {
    return PubButton;
  }
}
```

3. Pass the computed value to the pub-link component via the 'as'-prop.

```javascript
<pub-link :as="pubButton">Link</pub-link>
```
